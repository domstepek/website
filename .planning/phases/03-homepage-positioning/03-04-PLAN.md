**In src/styles/global.css:**

3. Replace the `:root` custom properties block with dark theme values:
   ```css
   :root {
     color-scheme: dark;
     --bg: #0a0a0a;
     --bg-elevated: #141414;
     --text: #e0e0e0;
     --muted: #8a8a8a;
     --border: #2a2a2a;
     --accent: #6fba7f;
     --accent-strong: #7fd48f;
     --focus: #e0a040;
     --shell-width: 72rem;
     --measure: 68ch;
     --font-mono: "Space Mono", ui-monospace, "SFMono-Regular", "SF Mono", Consolas, monospace;
     --step-0: clamp(1rem, 0.97rem + 0.18vw, 1.125rem);
     --step-1: clamp(1.5rem, 1.28rem + 1.1vw, 2.4rem);
     --step-2: clamp(2.1rem, 1.7rem + 2vw, 3.6rem);
   }
   ```
   Key changes: dark bg (#0a0a0a), light text (#e0e0e0), muted gray, dark border, green accent (terminal-like), orange focus ring, monospace font variable replacing --font-sans.

4. Update `body` to use `--font-mono` instead of `--font-sans`:
   ```css
   body {
     font-family: var(--font-mono);
   }
   ```

5. Update link styles for dark theme visibility:
   ```css
   a {
     color: var(--accent-strong);
     text-decoration-line: underline;
     text-decoration-thickness: 0.11em;
     text-underline-offset: 0.17em;
     font-weight: 600;
   }
   a:hover {
     color: var(--accent);
     text-decoration-thickness: 0.16em;
   }
   ```

6. Update `.site-header, .site-footer` to use dark elevated bg instead of light mix:
   ```css
   .site-header,
   .site-footer {
     border-block: 1px solid var(--border);
     background: var(--bg-elevated);
   }
   ```

7. Update `.skip-link` for dark theme (invert the text/bg relationship):
   ```css
   .skip-link {
     background: var(--text);
     color: var(--bg);
   }
   ```

8. Add the CRT scanline overlay styles AFTER the existing CSS (before media queries):
   ```css
   /* --- CRT scanline overlay --- */
   .crt-overlay {
     position: fixed;
     inset: 0;
     z-index: 9999;
     pointer-events: none;
     background: repeating-linear-gradient(
       to bottom,
       transparent 0px,
       transparent 2px,
       rgba(0, 0, 0, 0.15) 2px,
       rgba(0, 0, 0, 0.15) 4px
     );
     mix-blend-mode: multiply;
   }
   ```
   This creates subtle 2px horizontal scanlines across the entire viewport. The `pointer-events: none` ensures it never blocks interaction. The `aria-hidden="true"` on the HTML element ensures screen readers ignore it. The low opacity (0.15) keeps text readable.

9. Update `.site-main code` font-family to use the new variable:
   ```css
   .site-main code {
     font-family: var(--font-mono);
     font-size: 0.94em;
   }
   ```

10. Update `.note-page__body pre` background for dark theme consistency:

    (This is already using variables, just confirming it works with dark values.)
